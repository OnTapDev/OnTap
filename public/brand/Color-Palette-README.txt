OnTap Color Palette README
Overview

This document defines the official OnTap color palette and usage guidelines. Following these rules ensures a cohesive, professional, and visually consistent brand across web, mobile, and marketing materials. Designers and developers may use shades of these colors to establish hierarchy or visual emphasis, but do not stray far from this palette.

Brand Colors
1. Background

Charcoal

Hex: #1A1A1A

Usage: Primary background for web and mobile interfaces. Prefer dark to provide high contrast for text and accent colors. Optional lighter and darker shades may be used for layering or sections, but maintain overall tone.

2. Primary Text

Warm Sand

Hex: #B2A88A

Usage: Main body text, labels, and supporting information. Provides clear readability against dark backgrounds.

3. Titles / Headings

Warm White

Hex: #F3E7D3

Usage: Primary titles, headings, and important interface labels. Can use slightly lighter or darker shades for hierarchy but avoid introducing new colors.

4. Accent / CTA

Olive Gold

Hex: #7D7254

Usage: Highlights, call-to-action buttons, active states, or elements requiring user attention. Shades may be used for hover, pressed, or disabled states.

5. Secondary / Supporting Text

Warm Gold

Hex: #7D6854

Usage: Secondary text, placeholder text, subtle backgrounds, or muted elements. Can adjust shade slightly to indicate hierarchy without straying from palette.

Usage Guidelines

Hierarchy: Use Warm Gold for main titles, Warm White for body text, Olive Gold for accents, and Warm Sand for secondary/supporting text. Shades of these colors can indicate hierarchy.

Consistency: Avoid introducing colors outside of this palette. All variations must feel cohesive with the brand.

Contrast: Ensure sufficient contrast for readability, especially for text over Charcoal backgrounds.

Shading: Slightly darker/lighter variations of these colors can be used to indicate states or hierarchy, but never introduce entirely new tones.

Notes for Designers & Developers

Maintain the integrity of the palette; don’t introduce unrelated colors.

Use shades of the palette for hierarchy, hover states, and emphasis.

Keep all interface elements professional, readable, and visually balanced.

This palette is flexible enough for branding, UI, and marketing materials, but all adaptations should remain true to the original colors.
